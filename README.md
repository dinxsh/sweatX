## setup 

1. **Install Expo CLI:**
   ```bash
   npm install -g expo-cli
   ```

2. Clone the repository:

   ```git clone https://github.com/dinxsh/sweatx.git```


3. Install dependencies:

   ```cd sweatx
   npm install
   ```

4. Start the project:

   ```expo start```
